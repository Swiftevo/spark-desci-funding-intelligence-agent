## 執行摘要

這個項目提議建立一套以 LLM 為基礎的公共物品資助（PGF）申請評估方法，目標是降低資助輪的評審成本，讓更多資金可以直接流向具影響力的項目。團隊聲稱已完成研究設計，包括選定五個 LLM 模型、定義 RBTS 與平均分測試流程、估算 token 成本，以及找出五個過往資助輪作為研究對象。

不過，項目目前仍處於非常早期的階段，主要停留在研究設計，尚未有原型、試點結果或已發表成果。其核心主張是 LLM 可以可靠地取代或輔助人類評審，在高風險的資金分配決策中進行判斷；這個主張具有明顯的方法論與倫理風險，但申請資料尚未充分處理這些問題。

- 降低成本後，更多資金可以直接流向氣候、社群與社會影響項目。

## 跨項目比較

DSPJ-0003 是本輪中唯一直接針對 AI-based funding evaluation 的項目，因此具有獨特定位。DSPJ-0002（Narrative Audits for Public-Goods）與它互補，因為該項目會壓力測試項目敘事，這可能成為 LLM 評審的一種輸入訊號；但 DSPJ-0002 屬於 Social / Community 領域，且已有更多進展，例如已交付 narrative audit、hackathon 獲獎等。

DSPJ-0049（FunDeSci）和 DSPJ-0027（Spark DeSci）都是 funding infrastructure 項目，但沒有使用 AI 進行評估。與這些同輪項目相比，DSPJ-0003 的已展示進展最少，目前只有研究設計；同時技術風險最高，因為它依賴尚未在高風險資助決策中驗證過的 LLM 能力。

## 風險標記

- **高風險：LLM 在資助決策中的評估偏誤**  
  LLM 已知可能出現人口統計、語言和主題偏誤。在資助情境中，偏誤評估可能系統性地不利於某些社群或項目類型。申請資料沒有處理偏誤緩解。

- **高風險：核心假設未經驗證**  
  項目的核心主張是 LLM 可以可靠地評估資助申請，但申請中沒有提供實證支持，沒有試點數據、benchmark 或既有發表成果。

- **中風險：早期階段且沒有交付成果**  
  項目目前處於「研究設計已完成」階段，沒有原型、試點結果或已發表方法論。資助一個尚未展示執行能力的研究設計具有執行風險。

- **高風險：AI 取代人類評審的倫理問題**  
  在資源分配中以 AI 取代或輔助人類判斷，會引發問責、透明度與公平性的嚴重倫理問題。申請資料沒有討論這些問題，也沒有提出 safeguards。

- **中風險：缺乏 ground truth 驗證方法**  
  申請提到 RBTS 與平均分測試，但沒有說明如何將 LLM 輸出與人類評審決策比較，也沒有說明何種一致程度才算成功。

- **低風險：團隊資歷不透明**  
  團隊聲稱擁有 20 年以上 IT 經驗和 AI / PGF 專長，但沒有提供可驗證的具體資歷、發表成果或過往項目連結。
