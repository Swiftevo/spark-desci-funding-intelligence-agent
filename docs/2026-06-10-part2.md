### API Limitations

- Semantic Scholar results are real literature metadata, but not exhaustive.
- Rate limits apply. If available, set `SEMANTIC_SCHOLAR_API_KEY` for more stable access.
- Not all DeSci topics will have extensive literature coverage.
- AMiner integration remains planned for extended coverage when API access is available.
